### Prerequisites
- Flutter SDK 3.19.0+
- Android Studio / Xcode
- Node.js 18+
- Supabase account (for backend services)

### Setup Instructions

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd pharmaT
   ```

2. **Set up Flutter app**
   ```bash
   cd app
   flutter pub get
   flutter run
   ```

3. **Set up backend** (if using separate Node.js server)
   ```bash
   cd backend
   npm install
   npm start
   ```
